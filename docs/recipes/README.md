# Deploy recipes

Short, target-specific guides for getting bux running on different hosts.
Each recipe is self-contained — just enough to get from a blank box to
`sudo -iu bux && claude` in under 5 minutes.

**We'd love contributions!** If you got bux running on a provider / box type
that isn't documented here, open a PR with a short Markdown file covering
what you did. Good candidates: Hetzner, DigitalOcean, OCI free-tier,
Raspberry Pi, Mac mini over Tailscale, WSL, fly.io machines.
